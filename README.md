# 🌌 3D Portfolio Website

A stunning interactive 3D portfolio website built with Three.js and modern web technologies. Experience your portfolio in a whole new dimension!

## ✨ Features

### 🎨 **3D Visualization**
- Dynamic rotating cubes with orbital motion
- Particle system creating depth and atmosphere
- Professional lighting with neon glow effects
- Mouse-controlled camera tracking
- Smooth animations and transitions

### 🎭 **Modern UI/UX**
- Sleek navigation bar with smooth transitions
- Glassmorphism design with blur effects
- Neon cyan/blue aesthetic
- Responsive sections (Home, About, Projects, Contact)
- Beautiful hover effects and animations

### 📱 **Responsive Design**
- Fully responsive on mobile, tablet, and desktop
- Optimized performance across all devices
- Smooth scrolling experience

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML/CSS/JavaScript (for customization)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/23bcom20-art/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (http-server)
npx http-server
```

3. Visit `http://localhost:8000` in your browser

## 📝 Customization

### Update Personal Information
Edit `index.html` to add:
- Your name and bio in the Home section
- Your skills in the About section
- Your actual projects in the Projects section
- Your contact information in the Contact section
- Social media links

### Customize Colors
Edit `styles.css` to change:
- Neon colors (default: cyan #00ffff, green #00ff88)
- Background colors
- Glow effects intensity

### Modify 3D Objects
Edit `script.js` to customize:
- Number of cubes
- Cube sizes and colors
- Rotation speeds
- Orbital motion
- Particle system density
- Camera sensitivity

## 🌟 Tips for Best Experience

1. **High Performance Machine**: The 3D effects perform best on modern computers with dedicated GPUs
2. **Modern Browser**: Use the latest version of Chrome, Firefox, or Safari for best results
3. **Full Screen**: For an immersive experience, view in full screen mode
4. **Mouse Movement**: Move your mouse around to control the camera and see the parallax effect
